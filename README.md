# ShortAdvGame

// CLASSES //

Wizard:
- Advanced Attack Magic
- Basic Defensive Magic
- Teleportation Spell
- Uses a Wand

Cleric:
- Advanced Healing Magic
- Basic Attack Magic
- Basic Defense Magic
- Uses a Mace
- Little Extra Health

Warlock:
- Advanced Attack Magic
- Basic Healing Magic
- Mind Control Spell
- Uses a Staff

Eldritch Knight:
- Advanced Defensive Magic
- Basic Healing Magic
- Uses a Sword and Shield
- Extra Health

* Advanced = all spells
* Basic = only one (can choose)


// RACES //

Elf:
- Increased Movement Speed

Human:
- Little Extra Health
- Slightly Increased Movement Speed

Dwarf:
- Extra Health
