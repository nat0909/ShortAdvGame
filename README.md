# ShortAdvGame

// CLASSES //

Wizard:
- Advanced Attack Magic
- Basic Defensive Magic
- Relocate Spell (can be used on enemeis, allies, or oneself)
- Uses a Wand (can't melee)

Cleric:
- Basic Attack Magic
- Basic Defense Magic
- Advanced Healing Magic
- Uses a Mace (Decent amount of damage, range 1)
- Little Extra Health

Warlock:
- Advanced Attack Magic
- Basic Healing Magic
- Mind Control Spell
- Uses a Staff (Has a range of two but can only hit one enemy; doesn't do much damage, but it trips foes, reducing their movement next turn)

Eldritch Knight:
- Advanced Defensive Magic
- Basic Healing Magic
- Uses a Sword and Shield (Good amount of damage, range 1)
- Extra Health

* Advanced = all spells
* Basic = only one (can choose)


// RACES //

Elf:
- Increased Movement Speed

Human:
- Little Extra Health
- Slightly Increased Movement Speed

Dwarf:
- Extra Health

// Each senario is 50 long, if the player makes it past position 0 (the beginning) or 50, they exit the senario, but any remaining monsters will remain there

Senario:
_ _ _ _ + _ _ _ _ + _ _ _ _ o _ _ _ _ + _ _ _ _ o _ _ _ _ + _ _ _ _ + _ _ _ _ + _ _ _ _ + _ _ _ _ + 
Each symbol represents a square (each square takes one movement)
The o are places the character starts, and monsters can start on + or o, but not the o the character is occupying
